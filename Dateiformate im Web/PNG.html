Charakterisierung des Dateiformats PNG

Dateiformat PNG
Ausgeschriebener Name Portable Network Graphics
Medientyp

Informationsverlust
Kurzbeschreibung
PNG ist das meist verwendete verlustfreie Grafikformat im internet
Vorteile
  • Keine Lizenzgebüren
  • Verlustfreie Kompression
  • Unterstützung von Alpha-Transparenz
  • Durch integrierte Prüfsummen wird ermöglicht, Fehler in der Datei frühzeitig zu erkennen
Nachteile
  • Keine Möglichkeit zur Animation
  • Unterstützt das CMYK-Farbmodell nicht und ist deshalb nicht als vollständiger TIFF-Ersatz geeignet
  • Ermöglicht nicht das einfache Laden von Bildteilen
Einsatz im Web
Logos, Banner, Icons, Balken und so weiter
